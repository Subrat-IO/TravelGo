Welcome to **Travel Go** — a full-stack Airbnb-style web application where users can explore, list, and review beautiful travel destinations across the globe. Built with **Node.js**, **Express**, **MongoDB**, and **Bootstrap**, Travel Go aims to provide a seamless travel listing and booking experience.




---<img width="1894" height="908" alt="Screenshot 2025-08-13 005128" src="https://github.com/user-attachments/assets/c4077fbf-3187-4ad0-b46f-1f6b5dc5ebdd" />


## ✨ Features

- 🏡 Create, view, edit, and delete travel listings
- ⭐ Add and delete reviews with rating
- 📷 Image uploads for each listing
- 📱 Responsive design for mobile and desktop
- 🔍 Search, filter, and explore listings
- ✅ Data validation with Joi

---

## 🔧 Tech Stack

- **Frontend**: HTML, CSS, Bootstrap, EJS
- **Backend**: Node.js, Express.js
- **Database**: MongoDB, Mongoose
- **Templating**: EJS with ejs-mate
- **Validation**: Joi
- **Method Override**: For PUT/DELETE forms
